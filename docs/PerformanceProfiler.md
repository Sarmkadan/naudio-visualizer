# PerformanceProfiler

A lightweight utility for measuring and reporting the performance characteristics of code operations, including timing, call counts, and statistical distribution of execution durations. It is designed for scenarios where detailed performance diagnostics are required without the overhead of full-featured profiling tools.

## API

### `public string OperationName`
Gets the name of the operation being profiled. This value is set during construction and used in reports generated by `GetReport`.

### `public long TotalMs`
Gets the total accumulated time spent in the profiled operation across all calls, in milliseconds.

### `public int CallCount`
Gets the total number of times the profiled operation has been executed.

### `public long MinMs`
Gets the minimum execution time observed for the profiled operation, in milliseconds.

### `public long MaxMs`
Gets the maximum execution time observed for the profiled operation, in milliseconds.

### `public PerformanceProfiler(string operationName)`
Initializes a new instance of the `PerformanceProfiler` class with the specified operation name.

**Parameters**
- `operationName`: A string identifying the operation being profiled. Must not be null.

**Throws**
- `ArgumentNullException`: Thrown if `operationName` is null.

### `public TimingToken StartTimer()`
Starts a new timing measurement. The returned token must be disposed to record the elapsed time.

**Returns**
- A `TimingToken` that, when disposed, records the elapsed time for the operation.

### `public void RecordTime(long elapsedMilliseconds)`
Records a manually measured execution time for the profiled operation.

**Parameters**
- `elapsedMilliseconds`: The elapsed time in milliseconds to record.

**Throws**
- `ArgumentOutOfRangeException`: Thrown if `elapsedMilliseconds` is negative.

### `public double GetAverageTime()`
Calculates the average execution time of the profiled operation across all recorded calls.

**Returns**
- The average execution time in milliseconds, or `0` if no calls have been recorded.

### `public long GetTotalTime()`
Gets the total accumulated time spent in the profiled operation across all calls, in milliseconds.

**Returns**
- The total time in milliseconds.

### `public int GetCallCount()`
Gets the total number of times the profiled operation has been executed.

**Returns**
- The number of recorded calls.

### `public long GetMinTime()`
Gets the minimum execution time observed for the profiled operation, in milliseconds.

**Returns**
- The minimum time in milliseconds, or `0` if no calls have been recorded.

### `public long GetMaxTime()`
Gets the maximum execution time observed for the profiled operation, in milliseconds.

**Returns**
- The maximum time in milliseconds, or `0` if no calls have been recorded.

### `public long GetMedianTime()`
Calculates the median execution time of the profiled operation across all recorded calls.

**Returns**
- The median time in milliseconds, or `0` if no calls have been recorded.

### `public string GetReport()`
Generates a formatted performance report for the profiled operation.

**Returns**
- A string containing the operation name, call count, total time, average time, minimum time, maximum time, and median time.

### `public void Clear()`
Resets all recorded timing data, including call count, total time, min, max, and median values.

### `public TimingToken`
A disposable token returned by `StartTimer` that records the elapsed time when disposed. This type is not intended to be used directly.

### `public void Dispose()`
Releases all resources used by the `PerformanceProfiler` instance. This method is called automatically when the instance is used within a `using` statement or when garbage collected.
