# Security Policy

Please check the list of supported versions below:

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of naudio-visualizer seriously. If you discover a security vulnerability, we would appreciate it if you could report it to us responsibly.

Please report security issues using **GitHub's built-in private advisory feature** to disclose the vulnerability directly to the maintainers.

Alternatively, you can report vulnerabilities by contacting us via email at:
**rutova2@gmail.com**

### Acknowledgment
You should expect an acknowledgment of your report within **48 hours**. We will keep you updated on the progress of addressing the vulnerability.
